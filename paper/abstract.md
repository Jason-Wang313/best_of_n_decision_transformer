# Abstract

Source of truth: `paper/sections/abstract.tex`, included by `paper/main.tex`.

The LaTeX abstract frames the paper as a controlled mechanism study. Best-of-N raises tail pressure on the proxy score `S`; under out-of-support return conditioning, the selected proxy return rises while real utility `R` falls. The abstract reports the headline numbers (`S: 9.299 -> 9.468`, `R: 7.825 -> 7.381`) and the exact-law validation error (`0.00535`).
