Return-conditioned sequence policies turn desired return into an inference-time control input. A natural next step is Best-of-$N$: sample $N$ candidate trajectories and execute the one with the largest predicted or proxy return. This paper isolates a controlled failure mode in which the selected proxy return rises while real utility falls. The mechanism is not that Best-of-$N$ is intrinsically harmful, but that increasing $N$ shifts selection pressure toward the high-score tail; when that tail is outside offline support and the proxy underprices risk, higher selected return can be a false control signal. We give an exact finite-$N$, tie-aware law for the selected real-utility expectation, introduce a Return Fantasy Microscope and Tail Phase Diagram, and validate the law in a learned Decision Transformer-style synthetic study. In the out-of-support condition, naive Best-of-$N$ raises selected proxy return from $9.299$ to $9.468$ while real utility drops from $7.825$ to $7.381$ as $N$ grows from $1$ to $64$. The exact law matches Monte Carlo with maximum absolute error $0.00535$. In-support and anti-aligned controls behave as predicted, and likelihood or pilot-label repairs reduce the failure in this controlled environment. The result is a mechanism paper: it specifies when high return conditioning should be treated as evidence of improved utility, and when it should be audited as extrapolation.
