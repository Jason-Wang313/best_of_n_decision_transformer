# Method

Source of truth: `paper/sections/method.tex`, included by `paper/main.tex`.

The method section defines the controlled one-dimensional environment, the proxy reward `r_S(a) = a - 0.025 a^2`, the real reward with convex off-support penalties, the empirical support boundary (`q95 = 6.314`), the DT-style learned rollout model, the Return Fantasy Microscope, and all selection strategies.

The manuscript deliberately calls the model "DT-style" rather than benchmark-scale Transformer.
